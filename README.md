A personal **C# Reverse Engineering project with Entity Framework Core 9** (.NET 9).  
This lab demonstrates how to generate EF Core models from an existing database (Database‑First / Reverse Engineering).

---

## 📌 Project Overview
This project focuses on:
- ✅ Reverse‑engineering an existing database into C# models  
- ✅ Generating **DbContext** and **Entity classes** from database tables  
- ✅ Working with migrations and scaffolding tools  
- ✅ Practicing database‑first workflow with EF Core 9
